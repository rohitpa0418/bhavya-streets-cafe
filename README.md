# Bhavya Streets Cafe — Website

## 🚀 How to Host This Website

### Quick Start
1. **Unzip** the downloaded ZIP file
2. **Open** `index.html` in any browser — it works instantly
3. **Host** on Netlify, Vercel, GitHub Pages, or any web hosting

---

### 🖼️ How to Add Real Menu Images
1. Save your 3 menu images as:
   - `images/menu1.jpg`
   - `images/menu2.jpg`
   - `images/menu3.jpg`
2. Open `index.html`, find the Menu Gallery section
3. Replace the 3 Unsplash URLs with:
   - `images/menu1.jpg`
   - `images/menu2.jpg`
   - `images/menu3.jpg`
4. Also update the `menuImages` array in `js/script.js` the same way

---

### 📲 Update Social Media Links
Open `index.html` and search for:
- `@bhavyastreetscafe` → replace with your real Instagram handle
- Update Swiggy/Zomato URLs with your actual restaurant links

---

### 🌐 Hosting on Netlify (Free)
1. Go to [netlify.com](https://netlify.com) → Sign up free
2. Drag & drop your entire folder onto the Netlify dashboard
3. Done! Your site is live in seconds.
